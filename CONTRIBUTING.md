# Contributing to Clusterfudge

Thanks for your interest in Clusterfudge!

## Current Status

We are **not currently accepting external pull requests**. The project is under very active development and we don't have the bandwidth to review external contributions at this time.

This will change in the future — we'd love to collaborate with the community once things stabilize.

## Feature Requests

Have an idea for Clusterfudge? [Open a feature request](https://github.com/leonardaustin/clusterfudge/issues/new?template=feature_request.yml) on GitHub. Include:
- What you're trying to do (the problem, not just the solution)
- How you work around it today (if at all)
- Screenshots or examples if relevant

We read every request. Popular requests help shape the roadmap.

## Reporting Bugs

Found a bug? [Open a bug report](https://github.com/leonardaustin/clusterfudge/issues/new?template=bug_report.yml). Include:
- A clear description of the problem
- Steps to reproduce
- Your environment (OS, Clusterfudge version, Kubernetes version)
- Screenshots or logs if relevant
